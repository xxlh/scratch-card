ScratchCard demo

ScratchCard 是一个基于canvas实现“刮刮卡”功能的js库。

使用纯色填充蒙层：

已经刮开了0%


<div class="container">
  <p class="percent-tips">已经刮开了0%</p>
</div>
        

.container {
  position: relative;
  width: 800px;
  height: 450px;
}
.percent-tips {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate3d(-50%, -50%, 0);
  font-size: 36px;
}
.container canvas {
  position: absolute;
}
        

import ScratchCard from 'scratch-me'
const container = document.querySelector('.container')
const scratchCard = new ScratchCard(container, {
  mask: '#999',
  onScratch: function (percent) {
    document.querySelector('.container .percent-tips').textContent = '已经刮开了' + percent + '%'
  }
})
scratchCard.init()
显示代码

设置可刮区域

已经刮开了0%


  <div class="container">
    <p class="percent-tips">已经刮开了0%</p>
  </div>
          

.container {
  position: relative;
  width: 800px;
  height: 400px;
}
        

import ScratchCard from 'scratch-me'
const container = document.querySelector('.container')
const scratchCard = new ScratchCard(container, {
  mask: require('./bg.png'),
  scratchArea:{
    starX: 350,
    starY: 150,
    areaWidth: 100,
    areaHeight: 100
  },
  brushSize: 80,
  brushPress: 0.25,
  onScratch (percent) {
    document.querySelector('.container .percent-tips').textContent = '已经刮开了' + percent + '%'
  }
})
